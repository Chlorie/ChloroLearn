Detailed Description

This namespace contains most functions that work like operators, that is, taking some operands as input and returning an operand as a result.

Remarks
For more information on the actual Operand class, please refer to the corresponding header and documentation.

◆ categorical_cross_entropy()

Operand chloro::operators::categorical_cross_entropy ( Operand  predicted,
Operand  target 
)

Categorical cross entropy loss function.

Usually used as the loss function for classification tasks.

Parameters
predictedThe predicted value for the target one-hot vector. All components of predicted should be positive, and sum up to 1. Usually predicted vector is an output of softmax operation.
targetA scalar value array (shape 1), containing a non-negative integer representing the 0-based index of the value 1 in the target one-hot vector. Notice that the back-propagation process will not proceed to this branch, that is, the target won't be updated (in case it's an output of some operation).
Returns
A scalar value array containing the computed loss.
