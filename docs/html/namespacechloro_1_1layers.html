Detailed Description

Helper functions for adding neural network layers to a graph.

A neural network layer often consists of multiple nodes, for example, a dense layer (fully connected layer) consists of two variable nodes for weights and bias, two operator nodes for matrix multiplication and addition, and possibly another operator node for the activation function. These functions are to simplify the process of adding these nodes to a graph.
