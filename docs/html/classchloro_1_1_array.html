Detailed Description

template<typename T, typename = std::enable_if_t<std::is_arithmetic_v<T>>>
class chloro::Array< T, typename >

A flexible multi-dimensional generic array that supports basic operations and other functions like reshaping.

Specifically, this type is broadly used in the other parts of this library, more specifically, double version of this type.

Template Parameters
TType of data stored in the Array, should be an arithmatic type.
