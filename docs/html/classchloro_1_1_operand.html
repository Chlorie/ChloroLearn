Detailed Description

This class serves as a syntax tree for inserting multiple Operator objects together into the same graph.

For example, the expression x*(y+2) composes of two Operators, the addition and the multiplication, together with references to nodes x and y. Explicitly usage of this class is not recommended, except when you're trying to implement more operators by yourself. In such cases, please refer to the implementations in namespace chloro::operators. Explicit construction of this class is not recommended as well, since only rvalue references to this object is used in this library.

Join one or more operands together with an operator.

For example, for x+y*z, the plus operator joins two operands together: The operand containing only a NodeRef x, together with the operand containing an multiplication operator and two NodeRefs to y and z. For more information about this, see implementations in namespace chloro::operators.
