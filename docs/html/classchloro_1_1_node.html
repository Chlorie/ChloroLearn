Detailed Description

A class representing a node in the DAG flow graph. Users should not construct objects of this type manually, neither should users manually construct any of the four node content types to be mentioned below. Use the add_... methods of class Graph instead. Nodes can contain contents of types including Input, Constant, Variable and Operator. For more information on these four node types, please see their own documentation and implementation respectively.
